# Super Mario

---

## Descrição

O sistema consiste em um jogo simples inspirado no **Super Mario**, desenvolvido com **HTML, CSS e JavaScript puro**, focado em animações, lógica de jogo e manipulação do DOM.

A aplicação foi construída utilizando **js** seguindo boas práticas de organização e versionamento.
---

### Observações

- As rotas utilizam **JSON** para envio e retorno de dados.
- O parâmetro `:id` representa o identificador único do recurso.
- Os códigos HTTP seguem o padrão REST (`200`, `201`, `204`, `400`, `404`).
- Recomenda-se o uso de **Postman** ou **Insomnia** para testes da API.

 ##  Funcionalidades

- Personagem controlável (pular e agachar)
- Obstáculos animados (cano)
- Inimigo dinâmico (turtle)
- Sistema de colisão
- Sistema de pontuação em tempo real
- Sons e trilha sonora
- Tela de *Game Over*
- Botão de reinício

---

##  Controles

- `Espaço` = Pular
- `S` = Agachar
- Clique em **Restart** para reiniciar o jogo

---

## Tecnologias utilizadas

- **HTML5**
- **CSS3**
  - Animações com `@keyframes`
- **JavaScript (Vanilla JS)**
  - Manipulação do DOM
  - Áudio com `Audio()`
  - Game loop com `setInterval`
